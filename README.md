# 🍰 Sus Cookies — Website Katalog Kue Lebaran

Sus Cookies adalah website katalog kue modern yang menampilkan berbagai pilihan kue Lebaran seperti nastar, kastengel, putri salju, hingga bolu lapis Surabaya. Website ini dirancang dengan tampilan yang clean, responsive, dan user-friendly sehingga nyaman digunakan di berbagai perangkat.

## ✨ Fitur Utama

* 🧁 Menampilkan katalog kue lengkap dengan gambar, deskripsi, dan harga
* 🛒 Sistem keranjang belanja (cart)
* 🔢 Perhitungan total harga otomatis
* 💳 Simulasi checkout dan pembayaran
* 📦 Form input data pembeli (nama, nomor HP, alamat)
* 📱 Responsive design (mobile & desktop)

## 🎨 Teknologi yang Digunakan

* HTML5
* CSS3 (Responsive Layout & Styling)
* JavaScript (DOM Manipulation & Logic)

## 🚀 Tujuan Project

Website ini dibuat sebagai latihan pengembangan web frontend dengan fokus pada:

* Pembuatan UI/UX sederhana
* Implementasi interaksi pengguna
* Simulasi alur pembelian (e-commerce sederhana)

## 📌 Catatan

Project ini masih bersifat frontend-only (tanpa backend & database), sehingga data tidak disimpan secara permanen dan pembayaran masih berupa simulasi.
